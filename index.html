Readme

This Achievement Reorder Tool allows you to reorder the achievements in a .txt file. Simply drag and drop your file, and you will be able to rearrange the achievements before saving the changes back to the file.

Steps:

  • Drag and drop your achievement file or select it manually.
  • Reorder the achievements by dragging them.
  • Click "Save File" to download the updated [gameid-User].txt file.
  • Replace the original file in the RACache/Data directory with the reordered version.
