# Achievement Reorder Tool for RetroAchievements  

This project is intended for developers who are familiar with RetroAchievements files. The tool is designed to reorder achievements from the assets list, making it easier to manage and organize them.    

---

## How to Use  

### 1. Upload or Drag-and-Drop a File  
- Ensure you have the `[gameid]-User.txt` file with RetroAchievements data.  
- **Backup the original file** before making any changes.  
- Drag the file onto the drop area or select it using the file input.  

### 2. Reorder Achievements  
- The tool will parse the file and display the achievements in a list.  
- Reorder the achievements by dragging the items to the desired position.  

### 3. Save the Reordered File  
- Once you're satisfied with the new order, click the **Save File** button to download the reordered `[gameid-User].txt` file.  
- Replace the original file in the `RACache/Data` directory with the reordered version.  

---

## Features  
- Reorder achievements from the assets list of RetroAchievements files.
- Displays only the achievement assets, not the full text, for easier management.  
- Simple drag-and-drop interface for quick file uploading.
- Allows saving the reordered list as a new .txt file for easy integration with your project. 

## Prerequisites  
- A basic understanding of RetroAchievements files and the Emulators environment.  
- The `[gameid]-User.txt` file must be located in the `RACache/Data` directory.
